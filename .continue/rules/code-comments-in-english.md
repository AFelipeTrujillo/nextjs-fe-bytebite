description: Ensures all code identifiers, comments, and documentation within
  source files are written in English
alwaysApply: true
---

All code must be written in English: variable names, function names, component names, file names, and all comments must be in English. No Spanish or other languages in code or comments. User-facing UI text (labels, buttons, messages) can be in the user's preferred language as specified per project, but internal code and comments must always be in English.
